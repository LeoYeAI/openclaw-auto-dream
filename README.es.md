# 🌀 OpenClaw Auto-Dream

**Consolidación automática de memoria para agentes OpenClaw — como el sueño para tu IA.**

Inspirado en cómo el cerebro humano consolida recuerdos durante el sueño, Auto-Dream ejecuta periódicamente un "ciclo de sueño" que revisa los registros diarios de tu agente, extrae información valiosa y la consolida en la memoria a largo plazo.

## ✨ Características

- **🔍 Escaneo** — Revisa los registros de memoria de los últimos 7 días
- **🧠 Extracción** — Identifica decisiones, lecciones, personas, proyectos y tareas pendientes
- **🔗 Fusión** — Integra en las secciones estructuradas de MEMORY.md con deduplicación
- **✂️ Limpieza** — Archiva entradas obsoletas, fusiona las similares
- **🔒 Seguridad** — Nunca elimina los registros fuente, hace respaldo antes de cambios importantes

## 📦 Instalación

```bash
clawhub install openclaw-auto-dream
```
