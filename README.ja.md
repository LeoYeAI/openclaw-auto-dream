# 🌀 OpenClaw Auto-Dream

**OpenClawエージェントの自動記憶整理 — AIに「夢」を見させる。**

人間の脳が睡眠中に記憶を整理する仕組みからインスピレーションを得ています。Auto-Dreamは定期的に「夢サイクル」を実行し、エージェントの日次ログをレビューし、価値あるインサイトを抽出して長期記憶に統合します。

## ✨ 機能

- **🔍 スキャン** — 過去7日間のメモリログをレビュー
- **🧠 抽出** — 決定、教訓、人物、プロジェクト、未解決タスクを識別
- **🔗 統合** — MEMORY.mdの構造化セクションに重複排除しながら統合
- **✂️ 整理** — 古いエントリをアーカイブし、類似項目を統合
- **🔒 安全** — ソースログを決して削除せず、大規模な変更前にバックアップを作成

## 📦 インストール

```bash
clawhub install openclaw-auto-dream
```
