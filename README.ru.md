# 🌀 OpenClaw Auto-Dream

**Автоматическая консолидация памяти для агентов OpenClaw — сон для вашего ИИ.**

Вдохновлено тем, как человеческий мозг консолидирует воспоминания во время сна. Auto-Dream периодически запускает «цикл сновидений», который просматривает ежедневные журналы агента, извлекает ценные инсайты и объединяет их в долговременную память.

## ✨ Возможности

- **🔍 Сканирование** — Проверяет журналы памяти за последние 7 дней
- **🧠 Извлечение** — Определяет решения, уроки, людей, проекты и открытые задачи
- **🔗 Объединение** — Интегрирует в структурированные разделы MEMORY.md с дедупликацией
- **✂️ Очистка** — Архивирует устаревшие записи, объединяет похожие
- **🔒 Безопасность** — Никогда не удаляет исходные журналы, создаёт резервные копии перед крупными изменениями

## 📦 Установка

```bash
clawhub install openclaw-auto-dream
```
