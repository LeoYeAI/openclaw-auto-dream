**Consolidation automatique de la mémoire pour les agents OpenClaw — comme le sommeil pour votre IA.**

Inspiré par la façon dont le cerveau humain consolide les souvenirs pendant le sommeil, Auto-Dream exécute périodiquement un « cycle de rêve » qui examine les journaux quotidiens de votre agent, extrait les informations précieuses et les consolide dans la mémoire à long terme.

## ✨ Fonctionnalités

- **🔍 Analyse** — Examine les 7 derniers jours de journaux mémoire
- **🧠 Extraction** — Identifie décisions, leçons, personnes, projets et tâches en cours
- **🔗 Fusion** — Intègre dans les sections structurées de MEMORY.md avec déduplication
- **✂️ Élagage** — Archive les entrées obsolètes, fusionne les similaires
- **🔒 Sécurité** — Ne supprime jamais les journaux sources, sauvegarde avant les changements majeurs
